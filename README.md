### 🎲 추첨 방법
- 추첨 방법
    매월 말에 푼 문제를 푼 일수 만큼 상품에 응모한 뒤, 랜덤으로 추첨.
    따라서 문제를 많이 풀 수록 당첨될 확률이 높고, 한 문제만 풀더라도 당첨될 수 있습니다!
